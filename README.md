# i3blocks
Reimplementation of [my old i3blocks scripts][1] in go.  
Some hardcoded stuff, you'll figure it out.

```
$ ./i3blocks bandwidth
↓↑ 109 KiB/s 18 KiB/s

$ ./i3blocks loadavg
1.80 1.30 1.24 (5/1/0/93)

$ ./i3blocks ups
100% (29m55s)

```
